Lähestyin tehtävää kahdelta kantilta, jotta sain toteutettua mahdollisimman paljon tehtävänannosta.

Frontti nimisessä kansiossa on notepad++ kirjoitettu pieni www-sivu ja siihen liittyvä stylesheet.
Tässä käytetty html, css ja JavaScript. Logitus tällä puolella tapahtuu vain sivulla olevassa kentässä.

Visual_Studio nimisessä kansiossa on Visual Studio 2019 C# kirjoitettu ohjelma, joka kyselee variksia
ja harakoita ja täydentää valintojen mukaan logia tekstitiedostoon, josta aina valinnan jälkeen tulostetaan kuinka monta varista
ja harakkaa sinne on kirjattu.
